Default value : process.env.ADMIN_PASSWORD || 'password'

The password used to access the admin interface

clientPath
Default value : join(__dirname, '/../virtual-quiz')

The absolute path to the client

dbPassword
Default value : process.env.MONGODB_PASSWORD

Database password

dbUser
Default value : process.env.MONGODB_USER

Database username

secret
Default value : process.env.SECRET || 'test secret'

The secret used to sign all the javascript web tokens

start
Default value : process.env.START_DATE || '12/31/1969 7pm' || '12/12/2020 8pm'

The start of the competition. Pulls from environment variable or compares against the start of unix time (in CST and mostly for fun)

Default value : 60 * 1000

The amount of time before no answer is counted as an automatic incorrect guess, in milliseconds

Default value : 300 * 1000

The amount of time each break lasts, in milliseconds
